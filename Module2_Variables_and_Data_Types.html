Module 2: Variables and Data Types

JavaScript Variables

Definition: JavaScript variables ek box jaise hote hain jisme hum data store karte hain. Yeh ek naam hota hai jo value ko refer karta hai, jaise number, text, ya true/false value.

Why Important: Variables use hota hai jab hum data ko store, update, aur reuse karna chahte hain. Web apps mein user name, score, price, aur form values sab variables mein rakhte hain.

Syntax:

let name = 'Aman';
const pi = 3.14;
var age = 20;

Example 1:

let score = 10;
console.log(score);

Explanation: score variable banaya aur usme 10 store kiya. Fir console pe print kiya.

Example 2:

const name = 'Riya';
console.log('Student:', name);

Explanation: name ko const se banaya, jiska matlab value change nahi hogi. Yeh student ka naam store karta hai.

Example 3:

var count = 1;
count = count + 1;
console.log(count);

Explanation: count variable banaya aur uski value badhai, phir 2 print kiya.

Real Use Case: Shopping cart mein item quantity, game score, user input, aur page text show karne mein variables bahut useful hote hain.

Interview Questions:

  • Q1: let, const, aur var mein kya difference hai?
    A1: let aur const block scope hote hain, const change nahi hota. var
  • Q2: Kab const use karna chahiye?
    A2: Jab aapko value change nahi karni ho, jaise fixed number ya constant string, tab const use karein.

Summary: JavaScript Variables data store karte hain. let flexible hota hai, const fixed value ke liye, aur var purana tareeka hai. Variables se hum values ko reuse aur update kar sakte hain.
