JavaScript Operators

JavaScript me operators 6 type ke hote hain:

  • Arithmetic
  • Assignment
  • Comparison
  • Logical
  • Unary
  • Ternary

Operators Overview

Type Example Use
Arithmetic +, -, *, / Calculation
Assignment =, +=, -= Value assign
Comparison ==, ===, >, < Compare
Logical &&, ||, ! Condition
Unary ++, -- Single value
Ternary ? : Short if-else

Arithmetic


let a = 10, b = 5;
console.log(a + b);
console.log(a - b);
console.log(a * b);

Assignment


let x = 10;
x += 5;
x -= 2;

Comparison


console.log(5 == "5");
console.log(5 === "5");

Logical


console.log(true && false);
console.log(true || false);

Unary


let x = 5;
x++;
x--;

Ternary


let age = 18;
let result = (age >= 18) ? "Adult" : "Child";

Important Notes

  • == value check karta hai
  • === value + type check karta hai
  • && = AND, || = OR
  • Ternary = short if-else
