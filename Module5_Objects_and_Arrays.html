Module 5: Objects and Arrays

Theory

Is module mein objects, arrays, aur basic object-oriented concepts seekhenge.

Topics

  • Objects create aur access karna
  • Properties add, update, delete karna
  • Methods aur this keyword
  • Arrays create aur access karna
  • Common array methods: push, pop, shift, unshift, slice, splice
  • Iterating arrays
  • Constructors, prototypes, ES6 classes

Practice Tasks

  1. Ek object banayein student details ke saath aur properties console pe print karein.
  2. Ek array banayein, usme items add karein, remove karein, aur array methods test karein.
  3. Ek simple class banayein student ke liye aur ek object instance create karein.

Example

const student = {
  name: 'Sandeep',
  age: 18,
  subject: 'JavaScript',
  greet: function() {
    console.log('Hello ' + this.name);
  }
};
student.greet();

const fruits = ['apple', 'banana', 'mango'];
fruits.push('orange');
console.log(fruits);

class Person {
  constructor(name, age) {
    this.name = name;
    this.age = age;
  }
  info() {
    console.log(this.name + ' is ' + this.age + ' years old');
  }
}
const p = new Person('Asha', 20);
p.info();
