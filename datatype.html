JavaScript Data Types

Definition: Data types batate hain ki variable ke andar kis type ka data store hai.

Why Important: Data type se JavaScript ko pata chalta hai ki data par kaise operation karna hai.

Types of Data Types

  • Number
  • String
  • Boolean
  • Undefined
  • Null
  • Object
  • Array

1. Number

let num = 100;
console.log(typeof num); // number

2. String

let name = "Sandeep";
console.log(typeof name); // string

3. Boolean

let isActive = true;
console.log(typeof isActive); // boolean

4. Undefined

let x;
console.log(typeof x); // undefined

5. Null

let y = null;
console.log(typeof y); // object (JS bug)

6. Object

let user = {
    name: "Sandeep",
    age: 20
};
console.log(typeof user);

7. Array

let arr = [1, 2, 3];
console.log(typeof arr); // object

Example

let a = 10;
let b = "Hello";
let c = true;

console.log(a, b, c);

Practice Tasks

  1. Ek number variable banao aur type check karo
  2. Ek string aur boolean create karo
  3. Ek object aur array bana ke print karo

Script Example

function checkTypes() {
    let num = 50;
    let str = "JS";
    let bool = false;
    let arr = [1,2];
    let obj = {name:"Dev"};

    console.log(typeof num);
    console.log(typeof str);
    console.log(typeof bool);
    console.log(typeof arr);
    console.log(typeof obj);
}

checkTypes();
