Data Type Classification

Category Types Description
Primitive String, Number, Boolean, Undefined, Null Simple values store karta hai
Non-Primitive Object, Array, Function Complex / reference values store karta hai

String

let name = "Sandeep";

Number

let age = 18;

Boolean

let isLogin = true;

Undefined

let x;

Null

let data = null;

Object

let user = {name:"Sandeep"};

Array

let arr = [1,2,3];

Function

function test(){}

Important Notes

  • typeof se data type check kar sakte hain
  • null ka type "object" aata hai (JavaScript bug)
  • Array bhi internally object hota hai

console.log(typeof "Hello");   // string
console.log(typeof 10);        // number
console.log(typeof true);      // boolean
console.log(typeof null);      // object
console.log(typeof []);        // object
