Module 6: DOM Manipulation

\n

Theory

\n

Is page ka purpose aur module ki basic theory yahan diya gaya hai, jisse aap asan tareeke se samajh sakein.

Is module mein DOM ko access aur change karna seekhenge, saath hi events aur element creation bhi.

Topics

  • Document Object Model kya hota hai
  • Elements access karna (getElementById, querySelector)
  • Content aur attributes modify karna
  • Event listeners (click, mouseover, input)
  • Event propagation, preventDefault, event delegation
  • Elements create, append, remove, replace
  • JavaScript se styles modify karna

Practice Tasks

  1. Ek HTML page banayein jisme button click se text change ho.
  2. Input field ka value read karein aur page par display karein.
  3. Dynamic elements create karein aur DOM mein add karein.

Example

document.addEventListener('DOMContentLoaded', function() {
  const heading = document.querySelector('#title');
  const button = document.querySelector('#changeButton');

  button.addEventListener('click', function() {
    heading.textContent = 'JavaScript se text change ho gaya!';
    heading.style.color = 'blue';
  });
});
