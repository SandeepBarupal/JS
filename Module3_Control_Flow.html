Module 3: Control Flow

Theory

Is module mein conditional statements aur loops seekhenge, jo program ko decision lene mein madad karte hain.

Topics

  • if, else if, else
  • Ternary operator
  • switch statement
  • for, while, do...while loops
  • for...in aur for...of loops
  • break aur continue

Practice Tasks

  1. Ek conditional script likhein jo age check kare aur message print kare if/else se.
  2. Ternary operator use karke ek short condition likhein aur result console mein dekhein.
  3. Loop banayein jo 1 se 10 tak numbers console kare, aur break aur continue ka use karein.

Example

let age = 20;
if (age >= 18) {
  console.log('Aap vote de sakte ho');
} else {
  console.log('Aap abhi chhote ho');
}

let message = age >= 18 ? 'Adult' : 'Minor';
console.log(message);

for (let i = 1; i <= 10; i++) {
  if (i === 5) continue;
  console.log(i);
}
