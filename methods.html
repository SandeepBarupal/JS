Methods

JavaScript Methods

Definition: Methods wo functions hote hain jo kisi object ke andar define hote hain.

Why Important: Methods se object ke data par kaam kar sakte hain, jaise string ko uppercase karna ya array ko modify karna.

Syntax:

object.methodName();

Example 1:

let text = "hello";
console.log(text.toUpperCase());

Example 2:

let arr = [1, 2, 3];
arr.push(4);
console.log(arr);

Example 3:

let name = "Sandeep";
console.log(name.length);

Real Use Case: Data manipulate karna, strings change karna, arrays manage karna.

Interview Questions:

  • Q1: Method kya hota hai?
    A: Object ke andar function ko method kehte hain.
  • Q2: push() method kya karta hai?
    A: Array ke end me element add karta hai.

Summary: Methods object ke actions hote hain. String aur array ke methods bahut use hote hain.
