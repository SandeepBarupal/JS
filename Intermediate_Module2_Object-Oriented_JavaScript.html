Intermediate Module 2: Object-Oriented JavaScript

Theory

Yahan prototypes, ES6 classes, encapsulation, aur abstraction ko deeper level pe dekhenge.

Topics

  • Prototype chain aur inheritance
  • Object.create se object banana
  • ES6 class syntax aur inheritance
  • Static methods aur properties
  • Private/public methods
  • Getters aur setters
  • Encapsulation implement karna

Practice Tasks

  1. Prototype chain ek object banakar samjho.
  2. Class banayein aur inherit karne wala subclass create karo.
  3. Getter aur setter use karke data access control karo.

Example

function Person(name) {
  this.name = name;
}
Person.prototype.greet = function() {
  console.log('Hello ' + this.name);
};
const p1 = new Person('Sandeep');
p1.greet();

class Animal {
  constructor(type) {
    this.type = type;
  }
  static info() {
    console.log('Animals are living beings');
  }
}
class Dog extends Animal {
  constructor(name) {
    super('Dog');
    this.name = name;
  }
}
const pet = new Dog('Sheru');
console.log(pet.type, pet.name);
Animal.info();
