Is module mein asynchronous JavaScript, error handling, aur AJAX/Fetch API ka introduction hai.

Topics

  • Asynchronous programming samajhna
  • setTimeout aur setInterval
  • Promises aur async/await
  • try, catch, finally
  • XMLHttpRequest aur Fetch API
  • JSON data handle karna

Practice Tasks

  1. Simple setTimeout use karo aur message delay ke baad show karo.
  2. Ek promise banao jo 2 seconds ke baad resolve ho aur console mein message show kare.
  3. Fetch API use karke kisi public JSON API se data load karo aur console mein dekho.

Example

function delay(ms) {
  return new Promise(resolve => setTimeout(resolve, ms));
}

async function run() {
  try {
    console.log('Wait karo...');
    await delay(2000);
    console.log('2 second baad yeh message aaya');
  } catch (error) {
    console.error('Error:', error);
  }
}

run();
