Intermediate Module 5: Working with Data Structures

Theory

Is module mein Map, Set, nested data, deep copy, aur destructuring patterns cover karenge.

Topics

  • Map, Set, WeakMap, WeakSet
  • Objects vs maps/sets
  • Deep copying objects and arrays
  • Nested data structures
  • Cloning and merging objects
  • Destructuring arrays and objects
  • Default values, aliasing

Practice Tasks

  1. Map aur Set create karo aur unke methods use karo.
  2. Deep copy example likho with nested object.
  3. Destructuring aur default values ka example banao.

Example

const user = { name: 'Sandeep', details: { age: 18, city: 'Ramsinghpur' } };
const copy = JSON.parse(JSON.stringify(user));
copy.details.city = 'New City';
console.log(user.details.city, copy.details.city);

const map = new Map();
map.set('a', 1);
map.set('b', 2);
console.log(map.get('a'));

const { name, details: { age }, details: { city = 'Unknown' } } = user;
console.log(name, age, city);
