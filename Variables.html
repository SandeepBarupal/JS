Variables

JavaScript Variables

Definition: JavaScript variables ek box hote hain jisme aap data store kar sakte ho. Yeh ek naam hota hai jo number, text, ya true/false value ko refer karta hai.

Why Important: Variables tab zaroori hote hain jab aap data store karke use dhang se manage karna chahte ho. Web apps mein user inputs, scores, prices, aur dynamic text sab variables se handle hote hain.

Syntax:

let name = 'Aman';
const pi = 3.14;
var age = 20;

Example 1:

let score = 10;
console.log(score);

Explanation: score variable banaya aur usme 10 store kiya. Fir console pe print kiya.

Example 2:

const name = 'Riya';
console.log('Student:', name);

Explanation: name ko const se banaya, jiska matlab value change nahi hogi. Yeh student ka naam store karta hai.

Example 3:

var count = 1;
count = count + 1;
console.log(count);

Explanation: count variable banaya aur uski value badhai, phir 2 print kiya.

Real Use Case: Variables shopping cart quantity, game score, aur form fields jaise user input save karne mein use hote hain.

Interview Questions:

  • Q1: let, const, aur var mein kya difference hai?
    A1: let block scope hai aur value badal sakte hain. const block scope hai par value change nahi hoti. var
  • Q2: Kab const use karna chahiye?
    A2: Jab value fix ho aur change nahi karni ho, jaise constant number ya fixed string.

Summary: Variables data store karte hain. let flexible hai, const fixed value ke liye, aur var purana style hai. Variables se hum values ko reuse aur update kar sakte hain.

Example

// variable declaration
let score = 10;
const name = 'Sandeep';

score = 15; // let se value change kar sakte hain
// name = 'Ravi'; // const se value change nahi ho sakti

console.log(score, name);

Script Example

Ye script ek function banata hai, variable values set karta hai, aur console mein output dekhta hai.

function sandeep() {
    var a = 10;
    let b = 20;
    const c = 30;

    a = 40; // var redeclare / reassign allowed
    console.log(a + ' ' + b + ' ' + c);
}

sandeep();
