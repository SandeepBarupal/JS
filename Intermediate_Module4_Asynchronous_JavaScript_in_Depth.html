Intermediate Module 4: Asynchronous JavaScript in Depth

Theory

Is module mein promises, async/await, aur API calls ko deep level se dekhenge.

Topics

  • Promises creation aur chaining
  • Errors handling with Promises
  • Promise.all, Promise.race, Promise.allSettled
  • Async/await conversion
  • Error handling in async functions
  • Parallel vs sequential execution
  • Fetch API aur JSON
  • CORS basics

Practice Tasks

  1. Promise banao jo delay ke baad resolve ho.
  2. Multiple promises ko Promise.all ke saath run karo.
  3. API call use fetch karo aur JSON data display karo.

Example

function wait(ms) {
  return new Promise(resolve => setTimeout(resolve, ms));
}

async function run() {
  try {
    await wait(1000);
    console.log('1 second finished');
    const response = await fetch('https://jsonplaceholder.typicode.com/todos/1');
    const data = await response.json();
    console.log(data);
  } catch (error) {
    console.error('Error:', error);
  }
}
run();
