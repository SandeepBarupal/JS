Module 4: Functions

Theory

Is module mein functions, closures aur alag tarah ke function expressions ko dekhenge.

Topics

  • Function define aur call karna
  • Parameters, arguments, return values
  • Anonymous functions aur arrow functions
  • IIFE (Immediately Invoked Function Expressions)
  • Scope: local vs global
  • Closures ka introduction

Practice Tasks

  1. Ek function banayein jo do numbers add kare aur result return kare.
  2. Arrow function aur anonymous function dono se same task karein.
  3. Ek closure banayein jisme outer function ek value rakhe aur inner function use access kare.

Example

function add(a, b) {
  return a + b;
}
console.log(add(5, 7));

const multiply = (x, y) => x * y;
console.log(multiply(3, 4));

const counter = (function() {
  let count = 0;
  return function() {
    count++;
    console.log('Count:', count);
  };
})();

counter();
counter();
