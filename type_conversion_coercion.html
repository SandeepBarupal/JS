Type Conversion & Type Coercion

JavaScript me data type change karne ke 2 tarike hote hain:

  • 1. Type Conversion (Explicit)
  • 2. Type Coercion (Implicit)

Difference Table

Type Meaning Control
Type Conversion Manually type change karna Developer ke control me
Type Coercion Automatic type change JavaScript automatically karta hai

1. Type Conversion (Explicit)

String to Number


let x = "10";
let num = Number(x);
console.log(num); // 10

Number to String


let x = 10;
let str = String(x);
console.log(str); // "10"

Boolean Conversion


let val = 1;
console.log(Boolean(val)); // true

parseInt / parseFloat


parseInt("10.5");   // 10
parseFloat("10.5"); // 10.5

2. Type Coercion (Implicit)

String + Number


"5" + 2  // "52"

Number - String


"5" - 2  // 3

Boolean Conversion


true + 1  // 2
false + 1 // 1

Equality (== vs ===)


5 == "5"   // true
5 === "5"  // false

Important Notes

  • == value check karta hai (type ignore)
  • === value + type dono check karta hai
  • + operator string banata hai
  • - operator number convert karta hai

console.log(typeof Number("10")); // number
console.log(typeof ("5" + 1));    // string
console.log(typeof ("5" - 1));    // number
