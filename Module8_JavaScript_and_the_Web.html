Theory

\n

Is page ka purpose aur module ki basic theory yahan diya gaya hai, jisse aap asan tareeke se samajh sakein.

Is module mein forms, browser storage, aur ES6+ features ke basics seekhenge.

Topics

  • Form validation aur submission handling
  • localStorage aur sessionStorage
  • JSON parsing aur stringify
  • ES6 features: template literals, destructuring, spread/rest
  • Modules (import/export)

Practice Tasks

  1. Form create karo jisme user name aur email enter kare aur submit pe result show ho.
  2. localStorage mein user data store karo aur page reload par wapas load karo.
  3. Template literals aur destructuring use karke simple object data print karo.

Example

const user = { name: 'Sandeep', age: 18 };
const { name, age } = user;
console.log(`Name: ${name}, Age: ${age}`);

localStorage.setItem('user', JSON.stringify(user));
const saved = JSON.parse(localStorage.getItem('user'));
console.log(saved);
