Intermediate Module 3: Advanced Array Methods

Theory

Is module mein array iteration, transformation, sorting, aur immutability seekhenge.

Topics

  • map, filter, reduce
  • some, every, find, findIndex
  • Array sorting aur custom compare
  • Multi-dimensional arrays aur flattening
  • Immutable array/object patterns
  • concat, slice, spread operator

Practice Tasks

  1. Array ko map se transform karo aur filter se filter karo.
  2. reduce use karke total ya summary calculate karo.
  3. Immutable patterns se array copy aur modify karo.

Example

const numbers = [1,2,3,4,5];
const squared = numbers.map(n => n*n);
const evens = numbers.filter(n => n % 2 === 0);
const sum = numbers.reduce((acc, n) => acc + n, 0);
console.log(squared, evens, sum);

const nested = [[1,2],[3,4]];
const flat = nested.flat();
console.log(flat);

const newNumbers = [...numbers, 6];
console.log(newNumbers);
