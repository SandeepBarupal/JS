Naming System

Theory

Yeh page JavaScript aur programming ke liye variable naming ke best practices batata hai.

1. Meaningful name use karo

Aise variable names choose karo jo value ya purpose ko clearly batayein.

let userName = 'Sandeep';
const maxItems = 10;
let averageScore = 84.5;

2. Consistent style follow karo

JavaScript mein usually camelCase use hota hai variables aur functions ke liye.

let firstName = 'Asha';
let totalCount = 0;
function calculateTotal() {
    // ...
}

3. Single-letter names avoid karo

Short names sirf loop counters ya temporary values ke liye use karo.

for (let i = 0; i < items.length; i++) {
    const item = items[i];
    // ...
}

4. Constants ke liye const use karo

Jab value change nahi hogi, tab const ka use karo.

const TAX_RATE = 0.07;
const API_URL = 'https://api.example.com';

5. Naming system aur case styles

Common naming styles JavaScript aur general code ke liye:

  • Camel Case (JS mein most used): let userName = "Sandeep";
  • Pascal Case (class/constructor ke liye): let UserName = "Sandeep";
  • Snake Case: let user_name = "Sandeep";
  • Upper Case (constants): const PI = 3.14;

6. Reserved words avoid karo

JavaScript reserved keywords jaise var, function, ya class names ke liye mat use karo.

7. Good naming ke examples

let shoppingCart = [];
let totalPrice = 0;
let isLoggedIn = false;

function updateCart(item) {
    shoppingCart.push(item);
}

const maxUsers = 100;
